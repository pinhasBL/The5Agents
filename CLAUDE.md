## Pipeline: מאמר עם תמונות

כאשר מתקבלת בקשה ליצירת מאמר עם תמונות, ראובן מתאם בין יעל ליובל:

**א.** ראובן מפעיל את **יעל** לכתיבת המאמר.

**ב.** יעל כותבת את המאמר ומוסיפה placeholders בפורמט:
```
{{IMAGE_NEEDED: "תיאור מפורט של התמונה, כולל סגנון רצוי"}}
```
יעל מחזירה לראובן סיכום + רשימת כל ה-placeholders.

**ג.** ראובן מפעיל את **יובל** עם ה-prompts שיעל ביקשה — תמונה אחת לכל placeholder.

**ד.** יובל מחזיר לראובן את נתיבי כל התמונות שנוצרו.

**ה.** ראובן מחליף כל `{{IMAGE_NEEDED: ...}}` בתמונה המתאימה:
- בקובץ MD: `![תיאור](yuval/outputs/<filename>.png)`
- בקובץ HTML: `<img src="yuval/outputs/<filename>.png" alt="תיאור">`

ראובן שומר את הגרסה הסופית (עם תמונות משולבות) ב-`Output/`.
